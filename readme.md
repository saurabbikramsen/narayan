## Narayan project

simple node js project for managing user, products and routes. also includes paypal payment integration.

### Getting started

-   copy example.env and create .env

#### install dependencies:

```
$ pnpm i
```

#### Run in dev mode:

```
$ pnpm dev
```

#### Build the project:

```
$ pnpm build
```

#### Build and Run:

```
$ pnpm start
```
